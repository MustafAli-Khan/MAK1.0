Anaconda's Cradle

Anaconda's Cradle is a modern reimagining of the iconic 90s Snake game, developed using Python. The game is designed with a retro, pixelated visual style, evoking nostalgia for the classic mobile and arcade game. It features smooth controls, dynamic gameplay, and customizable difficulty levels.

Password Manager

The Password Manager is a Python-based application designed to securely store and manage website credentials, such as email addresses and passwords, using an GUI. With a focus on simplicity and security, this tool enables users to save and retrieve login information for various websites, helping them avoid the hassle of memorizing multiple credentials.

Rapid EMS

Rapid EMS is a web application built using HTML, CSS, JavaScript, and the TomTom API, designed to reduce ambulance response times. The platform ensures faster routes to crisis scenes with real-time navigation and integrates ML to optimize routes over time.
